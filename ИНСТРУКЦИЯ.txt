1.Для того что бы запустилось приложение, на компьютере должен быть установлен node.js

2.Установка зависимостей:
- Перейти в папку server в консоли cd server и вписать npm i , и так же сделать с client
- Переходим в папку cd client и вписать npm i

3.Запуск приложения:
- Перейти в папку server в консоли cd server вписать npm run dev
- Перейти в папку client в консоли cd client вписать npm run start
